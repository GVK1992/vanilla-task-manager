-Reminder Application

A simple and efficient Reminder Management Application built using Node.js, Express, and a lightweight HTML/CSS/JavaScript frontend.
Users can create, update, delete, and view reminders with an easy-to-use interface.

-Features

Add new reminders

View all reminders

Edit/update reminders

Delete reminders

API built with Express.js

Frontend built with HTML + CSS + Vanilla JavaScript

Organized project structure
